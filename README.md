# 中文场景专用 Typst 项目模板

本模板专为中文排版优化，适用于论文、报告、讲义等各类中文文档编写。

## 环境准备

### 安装 Typst
请先安装 [Typst](https://typst.app/)。确保使用最新稳定版本以获得最佳兼容性。

### 字体配置
模板使用以下字体，请按需安装：

- **英文字体**：`Maple Mono NF`（用于代码和等宽文本）  
- **中文字体**：  
  - 思源宋体（Source Han Serif）  
  - 思源黑体（Source Han Sans）  

> 💡 建议安装完整字重（如 Light、Regular、Bold 等），以避免 Typst 回退到伪粗体或伪斜体，确保排版质量。

### 编辑器推荐（VSCode）
为获得最佳开发体验，推荐在 VSCode 中安装以下插件：

- **Tinymist Typst**：提供语法高亮、智能补全、实时预览等语言支持  
- **Typst Math**：方便查看和输入数学符号与公式  
- **Claude Code**（可选）：集成 AI 辅助编写功能  

### 依赖管理
模板中引用的第三方 Typst 库（如 `@preview/...`）将自动从网络下载。请确保编译环境具备联网能力。
